Download game

Bước 1

Tải file game về từ nút bấm download client ở trên. Nếu muốn chơi chế độ nhiều người chơi thì bấm thêm nút Download server

Bước 2

Cài đặt file ShipInvaders.exe đã được cài

Bước 3

Vào trong thư mục ShipInvaders ở vị trí được chọn khi cài đặt trong bước 2. Chạy file main.exe

Bước 4 (Nếu chơi chế độ nhiều người)

Người dùng tải và cài đặt python vào máy. Cách tải python

Bước 4 (Nếu chơi chế độ nhiều người)

Vào trong thư mục chứa file server_game.py được tải. Bấm vào ô địa chỉ, gõ "cmd" rồi bấm Enter.

Bước 5 (Nếu chơi chế độ nhiều người)

Khi này, 1 cửa sổ console màu đen sẽ được mở lên. Người dùng nhập "python server_game.py" rồi bấm Enter

Bước 6 (Nếu chơi chế độ nhiều người)

Sau đó, màn hình sẽ xuất ra 1 địa chỉ IP. Ví dụ: 192.168.1.12. Người chơi nhập địa chỉ này vào ô Type IP sau đó bấm các nút chế độ chơi theo ý muốn
